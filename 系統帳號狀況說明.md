### 🎯 解決方案

客戶有以下方式獲得測試帳號：

## 方案一：自助註冊申請 ⭐ (推薦)

### 立即可用 - 3分鐘完成
1. 訪問系統 → 點擊「申請新帳號」
2. 填寫註冊表單（姓名、Email、密碼、角色）
3. **系統自動顯示明文帳密** → 立即截圖保存
4. 檢查郵箱完成驗證 → 使用保存的帳密登入

### 系統會顯示：
```
🔐 請保存您的帳號資訊
帳號: your-email@example.com
密碼: your-password-123  ← 明文顯示
角色: 系統管理員
```

## 方案二：技術人員手動創建

### 需要 Supabase 管理權限
1. Supabase Dashboard > Authentication > Users
2. 手動創建 Auth 用戶 (記得勾選 Auto Confirm)
3. 執行 SQL 創建對應的 profiles 記錄

## 📊 管理員功能

登入管理員帳號後，可以：
- **權限管理 > 帳號註冊記錄** 查看所有申請的帳號
- **搜尋、匯出、管理** 所有測試帳號
- **查看密碼** (支援顯示/隱藏)
- **批量匯出 CSV** 帳號清單

## 🚀 建議操作流程

### Step 1: 立即申請管理員帳號
```bash
# 建議申請資訊
姓名: 測試管理員  
Email: admin-test@yourdomain.com  # 使用真實信箱
密碼: admin123456                # 簡單好記
角色: 系統管理員                  # 最高權限
```

### Step 2: 申請其他角色帳號  
```bash
# 輔導老師測試帳號
Email: coach-test@yourdomain.com
密碼: coach123
角色: 輔導老師

# 操作人員測試帳號  
Email: operator-test@yourdomain.com
密碼: operator123
角色: 執行單位操作人員
單位: 測試單位 (TEST001)
```
