# 客戶需求完整核實清單

## 根據最新對話截圖整理的需求

### 一、撥付進度功能

#### 1. 撥付進度表格欄位（✅ 已有基礎，需修改）
- **撥款期數**：第一期、第二期、第三期、第四期
- **公文寄出日期**：日期選擇器 ✅ 已實現
- **入帳日期**：日期選擇器 ✅ 已實現

#### 2. 公文寄出日期前加檢核文件確認（❌ 需新增）
**第一期要檢核的文件：**
1. 補助契約書（一式四份，需用印）
2. 切結書
3. 第一期款收據

**第二期要檢核的文件：**
1. 修正對照表
2. 修正計畫書
3. 自行檢核表
4. 第二期款收據

**第三期要檢核的文件：**
1. 期中成果報告書
2. 第一期與第二期款經費明細表
3. 第三期款收據

**第四期要檢核的文件：**
1. 全案期末成果報告書
2. 全案執行經費明細表
3. 第三期與第四期款經費明細表
4. 第四期款收據

#### 3. 文件狀態選項（❌ 需修改）
- 未上傳
- 已收到
- 已退回
- 已完成

#### 4. 文化部收訖狀態選項（❌ 需修改）
- 未收到
- 已收到
- 待補件
- 已退回
- 已完成

#### 5. 在勾選已退回或待補件時，可以寫一下備註（❌ 需新增）
- 說明為何退件或要補什麼

---

### 二、輔導紀錄功能

#### 1. 新增表單缺少欄位（❌ 需新增）

- **訪視照片上傳**：只傳一張（至少四張）

#### 2. 要能看得出是誰選的狀態（❌ 需新增）
- 記錄是誰修改了狀態

---

### 三、計畫基本資料表

#### 1. 單位基本資料表欄位（根據 Excel 截圖）
- 計畫編號（由我們設定）
- 計畫名稱（文字）
- 執行單位（文字）
- 執行年度（文字）
- 計畫期程（文字，可由我們設定）
- 類別（勾選）：原鄉文化行動計畫類 / 都市文化行動計畫類
- 計畫代表人：姓名、職稱、電話、手機、email（姓名、職稱、和email必填，電話不一定要填）
- 計畫聯絡人：姓名、職稱、電話、手機、email（姓名、職稱、和email必填，電話不一定要填）
- 立案地址（文字）
- 聯絡地址（文字）
- 實施地點：原鄉（接文字）/ 都市（接文字）
- 申請金額（數字）
- 核定金額（數字）
- 輔導委員：姓名、職稱、電話、手機、email（姓名、職稱、和email必填，電話不一定要填）
- **主責人員**：姓名、職稱、電話、手機、email ❌ 需新增

### 六、輔導紀錄（根據 Word 文件格式）

#### 1. 基本資訊
- 受輔導單位（自動帶入）
- 計畫名稱（自動帶入）
- 輔導地點
- 輔導次數（第 X 次）
- 輔導方式：實地訪視 / 視訊 / 電話 / 其他（請說明）
- 填寫人
- 輔導日期
- 輔導時間（開始時間 - 結束時間）

#### 2. 出席人員（勾選）
- 輔導委員（自動帶入）
- 主責人員（自動帶入）
- 計畫代表人（自動帶入）
- 計畫聯絡人（自動帶入）
- 其他人員（文字輸入）

#### 3. 整體訪視結果
- 計畫執行進度：進度超前 / 符合進度 / 進度落後（改進策略）
- 計畫執行內容：進度超前 / 符合進度 / 進度落後（改進策略）
- 執行紀錄完善：進度超前 / 符合進度 / 進度落後（改進策略）
- 核銷憑證完備：進度超前 / 符合進度 / 進度落後（改進策略）

#### 4. 訪視內容
- 工作項目
- 訪視意見
- 進度總結

#### 5. 訪視照片
- 至少四張
